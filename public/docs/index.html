Introduction

Numstore is a high-performance database system designed for storing and managing numerical data efficiently. This documentation provides comprehensive information about installation, configuration, and usage.

Getting Started

To get started with Numstore, follow these simple steps to install and configure your first instance.

Prerequisites

  • Linux, macOS, or Windows operating system
  • At least 4GB of RAM
  • C compiler (GCC, Clang, or MSVC)

Installation

Linux

$ wget https://example.com/numstore-latest.tar.gz
$ tar -xzf numstore-latest.tar.gz
$ cd numstore
$ ./configure
$ make
$ sudo make install

Windows

C:\> numstore-installer.exe
C:\> numstore --version

macOS

$ brew install numstore
$ numstore --version

Usage

Once installed, you can start using Numstore through its command-line interface or programming APIs.

Basic Commands

# Start the Numstore server
$ numstore-server start

# Connect to the server
$ numstore-cli connect localhost:7777

# Create a new variable
> CREATE VARIABLE temperature TYPE float64

# Insert data
> INSERT INTO temperature VALUES (23.5, 24.1, 22.8)

API Reference

C Library

The Numstore C library provides low-level access to all database operations.

Function Description
ns_connect() Establish a connection to the database
ns_create_variable() Create a new variable
ns_insert() Insert data into a variable
ns_query() Query data from a variable
Warning: Always close your database connections properly to prevent resource leaks.

Python API

import numstore

# Connect to database
conn = numstore.connect('localhost:7777')

# Create variable
conn.create_variable('temperature', dtype='float64')

# Insert data
conn.insert('temperature', [23.5, 24.1, 22.8])

# Query data
data = conn.query('temperature', start=0, end=10)

# Close connection
conn.close()
